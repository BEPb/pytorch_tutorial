## How to install and run
____
### Clone the repository
 
```sh
$ cmd
$ git clone https://github.com/BEPb/pytorch_tutorial
$ cd pytorch_tutorial
```
 
### Install the necessary packages (Install dependencies)
```sh
$  pip install -r requirements.txt
```
